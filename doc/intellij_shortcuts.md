# IntelliJ IDEA 단축키 가이드 (레거시 코드 탐색용)
> **안내:** 10,000줄 이상의 방대한 메소드나 파일을 분석할 때 유용한 단축키들입니다. 길을 잃지 않고 빠르게 이동하는 데 집중했습니다.

## 1. 📂 대형 파일 탐색 (가장 중요)
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Ctrl` + `F12`** | **파일 구조 팝업** | 현재 파일의 모든 메소드/변수 목록을 보여줍니다. **이름을 입력하면 바로 필터링**되어 1만 줄 파일 내에서 특정 메소드 찾기에 최고입니다. |
| **`Alt` + `Up` / `Down`** | **이전/다음 메소드 이동** | 메소드 선언부 단위로 즉시 점프합니다. |
| **`Ctrl` + `G`** | **라인 번호 이동** | 특정 줄 번호로 즉시 이동합니다. |
| **`Ctrl` + `Alt` + `Left` / `Right`** | **뒤로 / 앞으로 가기** | 웹 브라우저처럼 바로 직전에 내가 머물렀던 위치로 돌아갑니다. |
| **`Ctrl` + `Shift` + `Backspace`** | **마지막 수정 위치** | 방금 코드를 고쳤던 위치로 즉시 되돌아갑니다. |
| **`F11`** | **북마크 설정/해제** | 나중에 다시 올 중요한 지점을 숫자로 찜해둡니다. |
| **`Shift` + `F11`** | **북마크 목록 보기** | 저장해둔 북마크들을 목록으로 보고 이동합니다. |

## 🔍 2. 코드 분석 및 추적
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Ctrl` + `B`** (또는 `Ctrl`+클릭) | **선언부로 이동** | 변수나 메소드가 정의된 곳으로 점프합니다. |
| **`Ctrl` + `Alt` + `B`** | **구현체로 이동** | 인터페이스의 실제 구현 코드로 바로 점프합니다. |
| **`Alt` + `F7`** | **사용처 찾기** | 이 변수나 메소드가 전체 프로젝트 중 어디서 쓰이는지 목록을 보여줍니다. |
| **`Ctrl` + `Shift` + `F7`** | **현재 파일 내 하이라이트** | 현재 파일 내에서 선택한 항목이 쓰인 곳을 모두 강조합니다. |
| **`Ctrl` + `Q`** | **빠른 문서 보기** | 커서를 옮기지 않고도 메소드 설명이나 파라미터를 팝업으로 봅니다. |
| **`Ctrl` + `P`** | **파라미터 정보** | 메소드 괄호 안에서 어떤 인자들이 필요한지 알려줍니다. |

## 🛠️ 3. 코드 접기 (화면 정리)
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Ctrl` + `.`** (마침표) | **코드 접기/펴기** | 현재 블록(if, for, method 등)을 접어서 숨깁니다. |
| **`Ctrl` + `Shift` + `+` / `-`** | **전체 펼치기 / 접기** | 파일 내 모든 코드를 접어서 전체 구조(메소드 목록)만 봅니다. |
| **`Ctrl` + `Alt` + `.`** | **선택 영역 접기** | 내가 선택한 부분만 일시적으로 접어서 숨깁니다. |

## ⌨️ 4. 편집 및 블록 선택
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Ctrl` + `W`** | **선택 영역 확대** | 단어 -> 문장 -> 블록 -> 메소드 순으로 선택 영역을 점점 넓힙니다. |
| **`Ctrl` + `Shift` + `W`** | **선택 영역 축소** | Ctrl + W의 반대로 선택 범위를 좁힙니다. |
| **`Ctrl` + `/`** | **주석 처리** | 현재 줄을 한 줄 주석(//)으로 처리/해제합니다. |
| **`Ctrl` + `Shift` + `[` / `]`** | **괄호 끝으로 이동** | 현재 위치한 블록의 시작 `{` 또는 끝 `}` 괄호로 점프/선택합니다. |
| **`Ctrl` + `Alt` + `L`** | **코드 정렬** | 들쭉날쭉한 인덴트를 인텔리제이 설정에 맞춰 정렬합니다. (Git diff 주의!) |

## 🚀 5. 프로젝트 전체 검색
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Shift` + `Shift`** (두 번 연속) | **전체 검색** | 파일, 클래스, 설정 등 가리지 않고 모든 것을 찾습니다. |
| **`Ctrl` + `Shift` + `F`** | **경로에서 찾기** | 프로젝트 전역에서 특정 텍스트를 검색합니다. |
| **`Ctrl` + `Shift` + `R`** | **경로에서 바꾸기** | 프로젝트 전역에서 특정 텍스트를 일괄 변경합니다. |
